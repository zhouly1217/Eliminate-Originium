南开大学大一 C++ 课程大作业 —— 经典打砖块游戏的 Qt 实现
项目介绍
本项目是南开大学大一 C++ 课程大作业，基于 Qt 框架复刻经典打砖块游戏，结合面向对象思想与几何矩阵运算，实现了完整的游戏逻辑、碰撞检测、多难度适配及视觉 / 音效反馈，是对 C++ 基础、Qt 图形编程、线性代数的综合实践。
核心特性
1. 多难度游戏模式
•	简单模式：3 行六边形砖块，小球初始速度 1.5；
•	中等模式：4 行六边形砖块，小球初始速度 2.0；
•	困难模式：自适应行数砖块，小球初始速度 2.5 且速度上限更高；
•	游戏中可通过1/2/3键切换难度，ESC键返回菜单。
2. 精准的碰撞检测系统
基于 Eigen 库实现矩阵几何变换，完成：
•	小球与边界（上下左右）、挡板、六边形砖块的碰撞判定；
•	碰撞后小球速度 / 角度的物理反馈、挡板边界碰撞限制；
•	砖块耐久度机制（碰撞后耐久度降低，为 0 时砖块消失）。
3. 动态游戏机制
•	小球碰撞砖块后逐步加速（带难度对应的速度上限）；
•	挡板通过A/D键控制左右移动，移动速度可配置；
•	游戏循环基于 Qt 定时器驱动，保证帧率稳定。
4. 视觉与音效优化
•	自定义纹理贴图（小球 / 挡板 / 砖块 / 背景），支持异形（圆形 / 六边形）裁剪绘制；
•	碰撞音效、菜单 / 游戏背景音的切换与循环播放；
•	抗锯齿绘制，保证图形显示流畅。
技术栈
•	编程语言：C++；
•	图形框架：Qt 6.11.0（QMainWindow、QPainter、QTimer、QSoundEffect、QMediaPlayer 等）；
•	数学库：Eigen；
•	设计思想：面向对象、回调函数、模块化设计。
运行说明
1.	开发环境：Visual Studio；
2.	依赖库：Eigen3（需配置项目 INCLUDEPATH）；
3.	编译运行：将项目文件导入 Qt Creator，配置 Eigen 路径后直接编译运行；
4.	游戏操作：
o	菜单页：1/2/3选择难度，任意键开始游戏；
o	游戏页：A/D控制挡板移动，1/2/3切换难度，ESC返回菜单。
项目结构
核心类设计体现面向对象思想：
•	GeometryBase：几何基类（抽象类），定义顶点更新、绘制、碰撞检测接口；
•	派生类：RectangleGeometry（挡板 / 边界）、CircleGeometry（小球）、HexagonGeometry（砖块）；
•	GameWindow：游戏主窗口，管理游戏状态、渲染、输入事件、碰撞检测调度；
•	Ball/Paddle/Brick：游戏核心实体类，实现各自的绘制、状态更新、碰撞逻辑。
开发心得
作为大一 C++ 课程作业，本项目综合运用了 C++ 面向对象、Qt 图形编程、线性代数（矩阵变换）等知识，从 0 到 1 实现了完整的游戏逻辑。在碰撞检测、几何绘制、音效管理等模块的开发中，加深了对 C++ 内存管理、接口设计、跨平台图形编程的理解，也体会到了数学在游戏开发中的实际应用。
